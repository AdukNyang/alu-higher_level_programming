# Python - Data Structures: Lists, Tuples

This project covers Python data structures including lists and tuples, with implementations of various functions to manipulate and work with these data types.

## Files

- `0-print_list_integer.py` - Function that prints all integers of a list
- `1-element_at.py` - Function that retrieves an element from a list
- `2-replace_in_list.py` - Function that replaces an element in a list
- `3-print_reversed_list_integer.py` - Function that prints integers in reverse order
- `4-new_in_list.py` - Function that replaces element in a copy of the list
- `5-no_c.py` - Function that removes 'c' and 'C' characters from a string
- `6-print_matrix_integer.py` - Function that prints a matrix of integers
- `7-add_tuple.py` - Function that adds two tuples
- `8-multiple_returns.py` - Function that returns tuple with string length and first character
- `9-max_integer.py` - Function that finds the biggest integer in a list
- `10-divisible_by_2.py` - Function that finds multiples of 2 in a list
- `11-delete_at.py` - Function that deletes item at specific position
- `12-switch.py` - Script that switches values of two variables
