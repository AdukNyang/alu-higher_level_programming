# Python - More Data Structures

This project contains Python scripts demonstrating advanced data structure operations including sets, dictionaries, and lambda functions.

## Files

- `0-square_matrix_simple.py` - Square matrix elements
- `1-search_replace.py` - Search and replace in lists
- `2-uniq_add.py` - Add unique integers
- `3-common_elements.py` - Find common elements
- `4-only_diff_elements.py` - Find different elements
- `5-number_keys.py` - Count dictionary keys
- `6-print_sorted_dictionary.py` - Print sorted dictionary
- `7-update_dictionary.py` - Update dictionary values
- `8-simple_delete.py` - Delete dictionary keys
- `9-multiply_by_2.py` - Multiply dictionary values
- `10-best_score.py` - Find best score
- `11-multiply_list_map.py` - Multiply list using map
- `12-roman_to_int.py` - Convert Roman numerals to integers

## Usage

```bash
python3 filename.py
```

## Requirements

- Python 3.x
- All files are executable
