# Python - Exceptions

This project covers Python exception handling concepts including try/except blocks, raising exceptions, and proper error handling techniques.

## Files

- `0-safe_print_list.py` - Function that safely prints x elements of a list
- `1-safe_print_integer.py` - Function that safely prints an integer
- `2-safe_print_list_integers.py` - Function that prints only integers from a list
- `3-safe_print_division.py` - Function that safely divides two integers
- `4-list_division.py` - Function that divides two lists element by element
- `5-raise_exception.py` - Function that raises a TypeError
- `6-raise_exception_msg.py` - Function that raises a NameError with a message
