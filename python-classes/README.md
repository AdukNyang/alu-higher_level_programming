# Python - Classes and Objects

This project introduces Object-Oriented Programming (OOP) concepts in Python through the implementation of a Square class.

## Learning Objectives

- Understanding OOP fundamentals
- Creating classes and objects
- Working with attributes (public, protected, private)
- Implementing methods and properties
- Using getters and setters
- Data encapsulation and information hiding

## Files

- `0-square.py` - Empty Square class
- `1-square.py` - Square class with private size attribute
- `2-square.py` - Square class with size validation
- `3-square.py` - Square class with area calculation
- `4-square.py` - Square class with property getter/setter
- `5-square.py` - Square class with print functionality
- `6-square.py` - Square class with position attribute

## Requirements

- Python 3.8.5
- pycodestyle compliant
- All files executable
- Proper documentation for modules, classes, and methods
